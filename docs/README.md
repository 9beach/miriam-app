# Miriam

iOS 달리기 기록 앱. HealthKit 의 러닝 데이터를 기간 · 코스 · 표준 거리 기준으로 정리합니다.

- 랜딩 페이지: <https://9beach.github.io/miriam-app/>
- TestFlight 베타: 위 페이지에서 초대 링크 확인

## 무엇인가요

HealthKit 의 러닝 기록을 기간(주 · 월 · 년 · 라이프타임), 코스, 표준 거리(5K · 10K · 하프 · 풀) 기준으로 묶어 추이, 순위, 수상 기록, 공유 카드를 한 앱에서 다루는 iOS 앱입니다.

자세한 기능은 [랜딩 페이지](https://9beach.github.io/miriam-app/)를 참고하세요.

## 피드백

- **TestFlight 내 피드백** — 스크린샷에 표시 + 디바이스 정보 자동 첨부 (가장 정확)
- **GitHub Issues** — [버그 제보](https://github.com/9beach/miriam-app/issues/new?template=bug.yml) · [기능 제안](https://github.com/9beach/miriam-app/issues/new?template=feature.yml)

이슈에 HealthKit 스크린샷을 첨부할 때 개인정보(이름, 위치, 시간 등)는 가려 주세요.

## 개인정보

Miriam 은 사용자의 건강·운동 데이터를 외부로 전송하지 않습니다. 러닝 기록은 디바이스 내부에만 저장되며 분석 SDK, 광고 SDK, 사용자 추적 도구를 사용하지 않습니다. 자세한 내용은 [랜딩 페이지의 개인정보 섹션](https://9beach.github.io/miriam-app/#privacy)을 참고하세요.
